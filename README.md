## 📌 프로젝트 소개  
학부생 시절 ‘가상 현실’ 강의의 개인 기말 프로젝트입니다.  
단순한 게임보다는 더 의미 있는 콘텐츠를 창출하고자 교육 프로그램으로 기획했습니다.  
Vuforia를 활용하여 AR을 구현하고 Unity를 사용하여 개발하였습니다.
